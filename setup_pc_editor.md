## Catatan Instalasi Komputer Editor

### 1. Instalasi Software

* [ ] Instal **Windows 10 LTSC**

* [ ] Aktivasi Windows

* [ ] Instal **Microsoft Office 2013**

* [ ] Aktivasi Office

* [ ] Instal **Adobe InDesign CS6**

* [ ] Instal **MathType**

* [ ] Instal **Google Chrome**

* [ ] Instal **Foxit PDF Reader**

* [ ] Instal **Hard Disk Sentinel**

* [ ] Instal **CrystalDiskInfo**

---

### 2. Setup Sistem

#### Jaringan

* [ ] Set **IP Address** sesuai jaringan kantor

#### Sharing Network

* [ ] Pin folder `\\renders`
* [ ] Pin folder `\\debian`
* [ ] Aktifkan **File Sharing**

#### Identitas Komputer

* [ ] **Rename PC Name** sesuai standar perusahaan

#### Storage

* [ ] **Partisi Hardisk**

  * C: Sistem
  * D: Data / Kerja Editor

#### Windows Explorer

* [ ] Atur **Quick Access**
* [ ] Tampilkan **File Extension**
* [ ] Tampilkan **Hidden File**

#### Optimasi Sistem

* [ ] Nonaktifkan startup yang tidak perlu
* [ ] Set **Performance Mode (Best Performance)**
* [ ] Bersihkan aplikasi bawaan yang tidak digunakan
* [ ] Pastikan sistem berjalan **ringan dan responsif**

---

### 3. Pengecekan Akhir

* [ ] Restart dan pastikan tidak ada error
* [ ] Cek koneksi ke server sharing
* [ ] Cek semua software bisa dibuka
* [ ] Pastikan penyimpanan normal (cek via Hard Disk Sentinel / CrystalDisk)
